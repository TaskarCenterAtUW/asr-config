# asr-config

The asr-config repository contains all configuration files related to the AVIV ScoutRoute mobile application.

Each configuration module is organized into dedicated folders to maintain clarity and scalability.

# Example: Force Update

The force-update folder includes all configuration files required for the Force Update feature, such as environment-based minimum version settings and related metadata.

This repository serves as a centralized configuration source for the AVIV ScoutRoute mobile app, ensuring consistency across environments and simplifying maintenance of feature-specific settings.
